1) Trinity_assembled.fasta:
Trinity de novo assembly: the transcriptome was assembled using the Trinity assembler from our RNA-seq data.

2) trinotate_annotation_report.tsv:
The full trinotate annotation report. 
The functional annotation data derived from the analysis of transcripts aligned against a custom QCYY01P.1.fsa_aa (Penaeus vannamei) protein database

3) Filtered_Trinity_assembled.fasta: 
Trinity De Novo Assembly filtered to obtain high quality contigs based on trinotate annotations.
